# Lottie Animations in Swift

Learning state machines, playback, animation controls (play, pause, stop) and haptics by replicating an existing app

---

## Quick start (install)

### Swift Package Manager (recommended)

1. In Xcode: *File > Add Packages...*.
2. Enter: `https://github.com/airbnb/lottie-ios`.

### CocoaPods

```ruby
pod 'lottie-ios', '~> 4.0'
```

Then `pod install`.

---


## Haptics integration

Common pattern: trigger a light haptic when a key animation event occurs (e.g., when an animation completes).
